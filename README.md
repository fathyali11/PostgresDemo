# Simple .NET 10 API with Docker (Postgres & Redis)

This is a simple CRUD API for managing products, designed as a learning project to explore **Docker**, **PostgreSQL**, and **Redis** integration within the **.NET 10** environment without the hassle of local installations.

## Project Goals
* Learn how to orchestrate a development environment using **Docker Compose**.
* Experience migrating from SQL Server to **PostgreSQL**.
* Implement **Distributed Caching** using Redis.
* Experiment with the new features of **.NET 10**.

## Tech Stack
* **.NET 10 Web API** (Latest LTS)
* **Docker & Docker Compose**
* **PostgreSQL** (Relational Database)
* **Redis** (Caching Layer)
